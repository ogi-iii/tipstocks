# tipstocks
Simple CR(U)D App for managing your favorite links!

# Build & Run
App will be built & run as docker-compose.

## \[Way 1\] simple setup

```bash
# as foreground
$ ./run.sh --build

# as background
# $ ./run.sh --build -d
```

## \[Way 2\] run without building container image

```bash
# as foreground
$ ./run.sh

# as background
# $ ./run.sh -d
```
